## 注意事項

- 変数名とかは変えていたりします。noとかxとかいう変数名は分かりにくくて生理的に無理🙅‍♂️
- 白熊は捻くれ者なので**えんどんの裏をかくようなまじやばプログラム**を書いていたりします。（俗にクソコードといいます）


**コピペは何も意味がありません。**　あくまで理解の補助に使いましょう。というより白熊が独創的なクソコードを書いている影響でコピペは先生にバレる可能性大です。
